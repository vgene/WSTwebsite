Brief Introduction: This questionare is implemented using MVC development model. Model: Person.java in app/Models, has attributes like id, name,age, gender, email. The Model part uses Ebean API and H2 in memory database to implement, so the SQL create and update part is hidden from user. View: Views use scala to implement. "index.scala.html", "main.scala.html" in app/views are two main view templates, person list is loaded dynamically onto page using AJAX technique. Controller: HomeController.java in app/controllers, has methods like addPerson, deletePersons, getPersons, provide API to add, query and delele person information in the database.
